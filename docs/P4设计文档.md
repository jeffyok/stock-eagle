## 一、数据库设计

### 1.1 t_portfolio（持仓表）

```sql
CREATE TABLE t_portfolio (
    id          BIGINT AUTO_INCREMENT PRIMARY KEY,
    code        VARCHAR(10)  NOT NULL COMMENT '股票代码，如 sh600519',
    name        VARCHAR(20)  NOT NULL COMMENT '股票名称',
    cost        DECIMAL(12,2) NOT NULL COMMENT '持仓成本（元/股）',
    quantity    BIGINT       NOT NULL COMMENT '持仓数量（股）',
    buy_date    DATE         NOT NULL COMMENT '买入日期',
    stop_loss   DECIMAL(12,2) NULL COMMENT '止损价（元）',
    take_profit DECIMAL(12,2) NULL COMMENT '止盈价（元）',
    note        VARCHAR(255) NULL COMMENT '备注',
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at  DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_code (code)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='持仓管理表';
```

### 1.2 t_risk_rule（风控规则表）

```sql
CREATE TABLE t_risk_rule (
    id            BIGINT AUTO_INCREMENT PRIMARY KEY,
    rule_key      VARCHAR(30)  NOT NULL UNIQUE COMMENT '规则键',
    rule_name     VARCHAR(50)  NOT NULL COMMENT '规则名称',
    rule_value    VARCHAR(50)  NOT NULL COMMENT '规则值',
    rule_type     VARCHAR(10)  NOT NULL COMMENT '类型：number/switch',
    description   VARCHAR(255) NULL COMMENT '规则说明',
    is_enabled    TINYINT(1)  DEFAULT 1 COMMENT '是否启用',
    updated_at    DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='风控规则配置表';
```

## 二、持仓管理模块设计

### 2.1 功能清单

| 功能 | 说明 | 优先级 |
|---|---|---|
| 持仓列表 | 展示所有持仓，实时拉取最新价，计算盈亏 | P0 |
| 新增持仓 | 表单录入股票代码、成本、数量、买入日期等 | P0 |
| 编辑持仓 | 修改成本、数量、止损止盈价 | P1 |
| 删除持仓 | 软删除（标记 deleted_at） | P1 |
| 实时行情 | AKShare 拉取当前价，计算浮动盈亏 | P0 |
| 止损止盈状态 | 根据实时价标记触发状态 | P0 |

## 三、风控模块设计

### 3.1 规则引擎逻辑

```
遍历持仓 → 获取实时价 → 逐条检查规则 → 生成预警列表 → 推送到飞书
```

### 3.2 检查规则明细

| 规则 | 检查逻辑 | 报警级别 |
|---|---|---|
| 止损触发 | 现价 ≤ 止损价 | 🔴 红色（立即） |
| 止盈触发 | 现价 ≥ 止盈价 | 🟢 绿色（提示） |
| 单票亏损 | (现价-成本)/成本 ≤ -X% | 🟡 黄色 |
| 单票仓位 | 市值/总市值 ≥ X% | 🟡 黄色 |
| 账户总亏损 | 总盈亏/总成本 ≤ -X% | 🔴 红色 |
| 账户回撤 | (当前市值-历史最高市值)/历史最高 ≤ -X% | 🔴 红色 |

### 3.3 风控页面布局

```
┌─────────────────────────────────────────────┐
│  🛡️ 风控规则配置                          │
├─────────────────────────────────────────────┤
│  规则列表（可编辑）：                      │
│  规则名称 | 当前值 | 类型 | 启用 | 操作  │
├─────────────────────────────────────────────┤
│  实时风控检查：                           │
│  [🔍 立即检查] 上次检查：2026-05-05 13:30│
├─────────────────────────────────────────────┤
│  当前预警（表格）：                       │
│  级别 | 股票 | 触发规则 | 当前值 | 阈值 │

```

### 4.2 推送流程

```
定时触发（每日 15:30）
    ↓
拉取市场数据（AKShare）
    ↓
拉取持仓数据 + 实时行情
    ↓
运行风控规则检查
    ↓
拉取龙虎榜 / 板块异动
    ↓
生成报告文本
    ↓
调用飞书 Webhook 推送
    ↓
记录推送日志（t_push_log）
```

### 4.3 飞书推送配置

在 `app/config.py` 中添加：

```python
FEISHU_WEBHOOK_URL = "https://open.feishu.cn/open-apis/bot/v2/hook/xxx"
```
